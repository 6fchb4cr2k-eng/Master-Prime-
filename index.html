Como conectar o OneDrive — 5 minutos

1. Acesse portal.azure.com com sua conta Microsoft

2. Busque "App registrations" → clique em "New registration"

3. Preencha:

Name: Master Prime App
Supported account types: Accounts in any organizational directory and personal accounts
Redirect URI: Single-page application (SPA) → http://localhost

4. Clique em "Register" → copie o Application (client) ID

5. Vá em "API permissions" → Add permission → Microsoft Graph → Delegated:

Files.ReadWrite.All
User.Read

6. Cole o Client ID no campo acima e salve
